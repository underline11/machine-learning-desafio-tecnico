##  Tecnologias e Conceitos Utilizados

- **Manipulação de DOM:** Criação dinâmica de elementos de tabela baseada em arrays.
- **Acumuladores:** Lógica de cálculo para somatório total da lista (`reduce` ou `forEach`).
- **Validação de Formulários:** Tratamento de tipos de dados (String, Int, Float).
- **CSS Flexbox:** Layout responsivo para facilitar o uso em dispositivos móveis.
